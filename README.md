This project demonstrates all major Django REST Framework (DRF) API approaches in a single Django application.
#Features
Function Based Views (FBV)
Class Based Views (APIView)
Mixins with GenericAPIView
Generic Class Based Views
ViewSets with Routers
Separate model & serializer for each API type
CRUD operations implementation
Tech Stack == Python , Django , Django REST Framework , SQLite
Purpose
This project is created for learning, exam preparation, and interview revision to understand and compare different DRF API styles in a structured way.


How to Run
python manage.py makemigrations
python manage.py migrate
python manage.py runserver

Conclusion
A beginner-friendly reference project to understand how the same CRUD logic is implemented using different DRF approaches.
